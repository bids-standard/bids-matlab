Indexing and querying a BIDS dataset
************************************

BIDS-Matlab allows you to index a raw or derivative BIDS dataset with :func:`bids.layout`,
and then query the content of that dataset with :func:`bids.query`.

The general API of these functions is detailed below.

For an example on how to use them, check out
`this jupyter notebook <https://github.com/bids-standard/bids-matlab/blob/dev/examples/BIDS-Matlab_basics.ipynb>`_.
